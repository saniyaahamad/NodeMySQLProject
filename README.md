# Node MySQL Project

## Description
This project demonstrates how to connect Node.js with MySQL using the mysql2 package. It performs various database operations such as creating a database, creating a table, inserting records, retrieving records, updating records, deleting records, altering the table, and dropping the table.

## Features
- Create Database
- Use Database
- Create Table
- Insert Records
- Select All Records
- Select Record by ID
- Update Record
- Delete Record
- Add Column
- Drop Column
- Drop Primary Key
- Drop Table

## Technologies Used
- Node.js
- MySQL
- mysql2
- JavaScript

## Installation

Install dependencies:

```bash
npm install mysql2
```

Run the project:

```bash
node assignment2.js
```
